霍夫曼编码(Huffman Coding)

霍夫曼编码-维基百科

简而言之，霍夫曼编码是无损数据压缩和权编码算法。所谓权编码，就是根据源符号出现的概率大小分配不同长度的编码。出现概率高的符号分配短码，概率低的符号分配长码，从而实现数据压缩。

霍夫曼编码通过构建一颗最优二叉树(一种带权路径长度最短的二叉树)来为符号分配编码。所谓树的带权路径长度，就是树中所有的叶结点的权值乘上其到根结点的路径长度。权值一般为符号出现频数。
